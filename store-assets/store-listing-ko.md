# Play Store 등록 정보 (한국어)

## 앱 이름 (최대 30자)
음식만안와요 — 가짜 배달 데모

## 짧은 설명 (최대 80자)
배달앱을 켜고 싶을 때 대신 여는 가짜 배달앱. 메뉴 고르고, 칼로리 아껴요.

## 자세한 설명 (최대 4000자)
🍱 음식만안와요는 진짜 배달앱처럼 메뉴를 고르고, 장바구니에 담고, 주문서 작성하고, 배달 추적까지 체험할 수 있는 데모 앱이에요.

단 한 가지 차이점: **실제 주문·결제·배달은 절대 발생하지 않습니다.**

📱 어떻게 쓰는 앱인가요?
"오늘 뭐 먹지?" 라는 생각이 들 때, 진짜 배달앱 대신 이 앱을 열어보세요. 메뉴를 둘러보고, 장바구니에 담고, 주문해보는 그 경험만으로도 충분한 만족을 얻을 수 있어요. 그리고 주문 완료 화면에서 **"내가 아낀 칼로리"** 를 확인하세요.

✨ 주요 기능
🍽️ 15개 가상 음식점, 40개 이상의 메뉴 (치킨, 라멘, 파스타, 마라탕, 초밥, 피자 등)
🛒 실제 배달앱과 동일한 장바구니 · 옵션 시스템 (맛 선택, 사이즈, 토핑)
🗺️ 토끼배달·거북이배달 두 가지 모드의 실시간 배달 추적 (가상)
🔥 주문 완료 후 "내가 아낀 칼로리" 표시
🎨 4가지 컬러 테마 (오렌지·보라·파랑·민트)
🌐 한국어 / English 즉시 전환

⚠️ 꼭 확인해주세요
• 본 앱은 완전한 데모 시뮬레이션입니다.
• 실제 주문, 결제, 배달은 절대 발생하지 않습니다.
• 모든 음식점·메뉴 정보는 가상의 데이터입니다.
• 입력하신 배달 정보(이름, 주소 등)는 서버에 저장되지 않습니다.
